# Dental Dome · Calculadora de presupuestos 2026

## Uso rápido
1. Abre `index.html` en Chrome/Safari/Edge.
2. Añade datos de clínica, doctor y paciente.
3. Añade conceptos de tarifa.
4. Usa “Copiar WhatsApp/email” o “Imprimir / PDF”.

## Subir a GitHub Pages
1. Crea una cuenta en GitHub.
2. Crea un repositorio llamado `dental-dome-presupuestos`.
3. Sube todos los archivos de esta carpeta.
4. En Settings → Pages → Deploy from branch → main → /root.
5. GitHub te dará una URL privada/pública según configuración del repositorio.

## Editar en VS Code
Abre la carpeta y modifica `index.html`.
Los precios están dentro de la variable `productos`.
